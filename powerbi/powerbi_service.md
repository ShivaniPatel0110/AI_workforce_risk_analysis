# Power BI Service Deployment

## Overview

The dashboard was published to Power BI Service for sharing and real-time access.

## Steps Performed

* Published report from Power BI Desktop
* Created workspace in Power BI Service
* Enabled data refresh
* Shared dashboard

## Features Used

* Power BI Cloud (Service)
* Dashboard sharing
* Scheduled refresh

## Outcome

Enabled cloud-based dashboard access and real-time reporting capabilities.
