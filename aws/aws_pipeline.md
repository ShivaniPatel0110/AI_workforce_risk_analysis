# AWS Data Pipeline

## Overview

This project simulates a real-world cloud data pipeline using AWS S3.

## Architecture

1. Raw dataset stored in AWS S3
2. Data processed and analyzed
3. Connected to Power BI for visualization

## Steps Performed

* Created S3 bucket
* Uploaded workforce dataset
* Connected Power BI to cloud data
* Simulated real-time data pipeline

## Tools Used

* AWS S3
* Power BI

## Outcome

Demonstrated ability to work with cloud-based data pipelines and integrate with BI tools.
